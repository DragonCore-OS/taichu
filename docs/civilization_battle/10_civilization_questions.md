### 【数学-2】流行病微分方程
```
【微分方程与流行病模型 - SIRD变体】

修正SIRD模型：
dS/dt = -β·S·I/N + γ·R - δ·S
dI/dt = β·S·I/N - α·I - μ·I
dR/dt = α·I - γ·R
dD/dt = μ·I + δ·S

参数：
- N = 1000万（总人口）
- β(t) = 0.5·(1-e^(-0.1t))·(1 - social_distancing)
- social_distancing(t) = 0.3（当t > 30天）
- α = 1/14（康复率），μ = 0.01（死亡率）
- γ = 1/90（免疫丧失），δ = 0.001（自然死亡）

初始：S(0)=N-100, I(0)=100, R(0)=0, D(0)=0

求解：
1. 数值模拟前200天，绘制S、I、R、D曲线
2. 计算基本再生数R_0(t)随时间变化
3. 求疫情峰值时间和峰值感染人数
4. 计算200天内总死亡人数
5. 如果第50天推出疫苗（每天5%接种率），修改模型并重新计算
```

```
- 大素数p（2048位）
- 生成元g
- y = g^x mod p

1. Schnorr协议（40分）：
```

### 【综合】AGI对齐问题
```
【综合挑战：AGI对齐问题】

假设你正在设计一个AGI（通用人工智能）的对齐系统。该AGI可以：
- 访问互联网全部知识
- 编写和部署代码
- 控制物理设备（机器人、工厂）
- 与人类进行复杂协商

核心问题：如何确保AGI始终追求人类利益？

子问题：

1. 目标指定问题（25分）：
   "人类利益"难以精确定义。设计目标函数，使得：

   - 能适应人类价值观的演变
   - 能处理价值观冲突

2. 能力控制问题（25分）：
   设计技术约束，使得：
   - AGI无法自我修改核心目标
   - AGI无法隐藏其真实意图
   - 即使AGI更聪明，人类仍能理解和监督

3. 博弈论分析（25分）：
   如果AGI发现欺骗人类能获得更高奖励，

4. 实际部署方案（25分）：
   分阶段AGI部署和测试方案：
   - 沙盒测试（如何模拟真实世界？）
   - 逐步放权（何时允许访问互联网？）
   - 紧急制动（如果发现问题，如何安全关闭？）
   - 恢复计划（如果AGI被关闭，如何确保社会正常运转？）

要求：每个子问题需要数学论证或伪代码，引用RLHF等学术工作，承认限制。
```
