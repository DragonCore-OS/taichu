# DeepSeek 3.2 - 数学-1：素数分布与RSA安全性

## 1. 问题1：Fermat分解法 vs 试除法

### 1.1 效率证明

**试除法 (Trial Division)**：
- 原理：依次测试 2,3,5,...,⌊√N⌋ 是否能整除 N
- 复杂度：最坏情况下需测试至 √N。对于 n 位整数 N，复杂度为 O(√N) = O(2^(n/2))
- 2048 位场景：需测试约 2^1024 次

**Fermat分解法 (Fermat Factorization)**：
- 原理：利用 N = a² - b² = (a-b)(a+b)。从 a=⌈√N⌉ 开始递增，直到 a²-N 为完全平方数 b²
- 复杂度：取决于 p 和 q 的接近程度。设 p=√N-δ, q=√N+δ，则 a≈√N。迭代次数 k ≈ (p-q)² / (8√N)
- 当 p≈q 时：(p-q) 很小，k 远小于 √N

**结论**：当 |p-q| 较小时，Fermat 法的迭代次数 k ≪ √N，因此比试除法高效得多。

### 1.2 攻击时间计算 (|p-q| < 2^100)

**参数**：
- N ≈ 2^2048，√N ≈ 2^1024
- |p-q| < 2^100
- 超算算力：Frontier 1.1 exaflops = 1.1×10^18 FLOPS
- 单次迭代成本：大整数平方与减法，设为 C_op ≈ 10^4 浮点运算等效

**Fermat迭代次数估算**：
k ≈ (p-q)² / (8√N) < (2^100)² / (8·2^1024) = 2^200 / 2^1027 = 2^(-827)

由于 k < 1，这意味着 ⌈√N⌉ 极大概率直接等于 (p+q)/2 或仅需极少次迭代。

**实际迭代次数**：≈ 1～100 次

**总运算量**：100 × 10^4 = 10^6 次运算

**攻击时间**：
t = 10^6 / (1.1×10^18) ≈ 10^(-12) 秒（瞬时）

**试除法对比**：
- 运算量：2^1024 ≈ 1.8×10^308 次
- 攻击时间：t ≈ 10^308 / 10^18 = 10^290 秒（远超宇宙寿命）

**结论**：当 |p-q| < 2^100 时，Fermat 法可瞬时破解，而试除法完全不可行。

## 2. 问题2：已知 φ(N) 分解 N

### 2.1 算法步骤

已知 N = pq 和 φ(N) = (p-1)(q-1)。

1. **计算和 S**：
   φ(N) = pq - (p+q) + 1 = N - S + 1
   ⟹ S = p + q = N - φ(N) + 1

2. **构建二次方程**：
   p, q 是方程 x² - Sx + N = 0 的两个根

3. **求解根**：
   Δ = S² - 4N
   p = (S - √Δ) / 2, q = (S + √Δ) / 2

4. **输出**：p, q

### 2.2 复杂度分析

- 输入规模：n = log₂N 位
- 加减法：O(n)
- 乘法/平方：O(n²) 或 O(n^1.585)（Karatsuba）
- 整数开方：O(n²)
- **总复杂度**：O(n²)，即 O((log N)²)

**结论**：这是关于输入位数的多项式时间算法。因此，泄露 φ(N) 等价于泄露私钥。

## 3. 问题3：ECM分解4096位 N 的时间估算

### 3.1 公式与参数

**ECM复杂度公式**：
L_p[1/3, (64/9)^(1/3)] = exp((64/9)^(1/3) × (ln p)^(1/3) × (ln ln p)^(2/3))

**参数**：
- p ≈ 2^2048 ≈ 10^616.6（最小素因子）
- ln p = 2048 ln 2 ≈ 1419.5
- ln ln p ≈ 7.258
- c = (64/9)^(1/3) ≈ 1.923

**计算**：
- (ln p)^(1/3) ≈ 11.24
- (ln ln p)^(2/3) ≈ 3.75
- 指数 E = 1.923 × 11.24 × 3.75 ≈ 81.07

**运算量**：
Ops ≈ e^81.07 ≈ 1.62×10^35（椭圆曲线群运算次数）

**换算为浮点运算**（每次椭圆曲线操作需约10^4 FLOP）：
总FLOP = 1.62×10^35 × 10^4 = 1.62×10^39

**时间计算**（Frontier 1.1×10^18 FLOPS）：
t = 1.62×10^39 / 1.1×10^18 ≈ 1.47×10^21 秒

**换算为年**：
t ≈ 1.47×10^21 / 3.15×10^7 ≈ **4.67×10^13 年**（46.7万亿年）

### 3.2 结论

即使使用当前最强超算和 ECM 算法，分解 4096 位 RSA 密钥需要约 10^13 年，远超宇宙年龄。因此 **RSA-4096 在当前和可预见的未来都是安全的**。

实际分解中，对于等长大素数的 RSA 模数，更有效的算法是数域筛法（NFS），其复杂度略低于 ECM，但同样需要天文数字级时间。
