# DeepSeek 3.2 - 代码-3：零知识证明系统（Schnorr协议）

## 系统概述

基于Schnorr身份识别协议，证明者向验证者证明其知道离散对数 x 满足 y = g^x mod p。

**安全参数**：
- 2048位大素数 p
- 生成元 g，阶为素数 q
- 128位安全级别

## 1. Schnorr协议（交互式）

```python
class SchnorrZK:
    def __init__(self, p, q, g):
        self.p = p
        self.q = q
        self.g = g
    
    def prove(self, x):
        """证明者生成承诺，等待挑战，返回响应"""
        # 第一阶段：承诺
        r = random_int(self.q)
        t = pow(self.g, r, self.p)
        
        # 第二阶段：接收挑战 e（由验证者生成）
        e = random_int(self.q)  # 实际应由验证者提供
        
        # 第三阶段：响应
        s = (r + e * x) % self.q
        return (t, e, s)
    
    def verify(self, t, e, s, y):
        """验证者检查 g^s == t * y^e mod p"""
        left = pow(self.g, s, self.p)
        right = (t * pow(y, e, self.p)) % self.p
        return left == right
```

### 安全性证明

**完备性**：
```
g^s = g^(r+ex) = g^r · (g^x)^e = t · y^e
```

**可靠性（提取器）**：
- 通过两次不同挑战 e₁, e₂ 得到 s₁, s₂
- x = (s₁ - s₂) · (e₁ - e₂)^(-1) mod q

**零知识**：
- 模拟器随机选择 e, s，计算 t = g^s · y^(-e)
- (t, e, s) 与真实分布不可区分

## 2. Fiat-Shamir变换（NIZK）

```python
    def nizk_prove(self, x, context=b''):
        """非交互式证明"""
        r = random_int(self.q)
        t = pow(self.g, r, self.p)
        
        # 挑战 e = Hash(g, y, t, context)
        y = pow(self.g, x, self.p)
        e = hash_to_int(self.g, y, t, context)
        
        s = (r + e * x) % self.q
        return (t, s)
    
    def nizk_verify(self, t, s, y, context=b''):
        """验证非交互式证明"""
        e = hash_to_int(self.g, y, t, context)
        left = pow(self.g, s, self.p)
        right = (t * pow(y, e, self.p)) % self.p
        return left == right
```

**防重放**：通过 context（时间戳、会话ID）确保证明唯一

## 3. 批量验证

```python
    def batch_verify(self, proofs, y_list, context=b''):
        """
        批量验证 n 个证明
        proofs: [(t, s), ...]
        y_list: [y1, y2, ..., yn]
        """
        n = len(proofs)
        
        # 生成随机系数 c_i
        coeffs = []
        for i in range(n):
            h = hashlib.sha256()
            h.update(str(i).encode())
            h.update(context)
            coeff = int(h.hexdigest(), 16) % self.q
            coeffs.append(coeff)
        
        # 计算左边：g^(sum c_i * s_i)
        sum_s = sum(coeffs[i] * proofs[i][1] for i in range(n)) % self.q
        left = pow(self.g, sum_s, self.p)
        
        # 计算右边：prod (t_i * y_i^e_i)^c_i
        right = 1
        for i in range(n):
            t, s = proofs[i]
            y = y_list[i]
            e = hash_to_int(self.g, y, t, context)
            term = (t * pow(y, e, self.p)) % self.p
            right = (right * pow(term, coeffs[i], self.p)) % self.p
        
        return left == right
```

**优化**：使用多指数算法（Pippenger），将n次模幂降为接近常数次
