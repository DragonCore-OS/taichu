## 1. 基础数学背景

- **群**：\( \mathbb{G} = \langle g \rangle \subseteq \mathbb{Z}_p^{\*}\)，其中  
  - \(p\) 是一个 **安全素数**（\(p = 2q+1\)，\(q\) 也是大素数），  
  - \(|g| = q\)（即 \(g\) 的阶是大素数 \(q\)），  
  - 离散对数问题在 \(\mathbb{G}\) 上具有 **≈128‑bit 安全** → 需要 \(|q| \approx 256\) 位，因而 \(|p| \approx 3072\) 位（NIST 推荐的 3072‑bit 模数）。

- **公开值**：\(y = g^{x} \bmod p\) （待证明知道的离散对数 \(x\)）。

- **哈希函数**：采用 **SHA‑256**（输出 256 bit），在 Fiat‑Shamir 中把 transcript 哈希成挑战 \(c \in \mathbb{Z}_q\)。

---

## 2. 交互式 Schnorr 证明（Prove / Verify）

| 角色 | 步骤 | 计算 |
|------|------|------|
| **Prover（P）** | 1. 选随机 \(r \stackrel{\$}{\leftarrow} \mathbb{Z}_q\)  <br>2. 计算承诺 \(a = g^{r} \bmod p\)  <br>3. 发送 \(a\) 给 Verifier | \(a = g^{r}\) |
| **Verifier（V）** | 1. 选随机挑战 \(c \stackrel{\$}{\leftarrow} \mathbb{Z}_q\)  <br>2. 发送 \(c\) 给 Prover | – |
| **Prover** | 1. 计算响应 \(s = r + c\cdot x \pmod q\)  <br>2. 发送 \(s\) 给 Verifier | \(s = r + c x\) |
| **Verifier** | 检查：\(g^{s} \stackrel{?}{=} a \cdot y^{c} \pmod p\) | 若等则接受 |

### 正确性
\[
g^{s}=g^{r+cx}=g^{r}\cdot(g^{x})^{c}=a\cdot y^{c}\pmod p .
\]

### 声音性（Soundness）
若伪造者不知道 \(x\) 但能让验证通过，则必存在两个不同的挑战 \(c\neq c'\) 使得同一承诺 \(a\) 能满足两个验证方程：
\[
g^{s}=a y^{c},\qquad g^{s'}=a y^{c'} .
\]
两式相除得到 \(g^{s-s'}=y^{c-c'}\) → 离散对数 \(x = (s-s')(c-c')^{-1}\pmod q\)，与假设矛盾。因此伪造概率 ≤ \(1/|q|\) ≈ \(2^{-256}\)（远低于 128‑bit 安全目标）。

### 零知识（Zero‑Knowledge）
构造 **模拟器**（Simulator）不需要知道 \(x\)：

1. 随机选取挑战 \(c \stackrel{\$}{\leftarrow}\mathbb{Z}_q\) 和响应 \(s \stackrel{\$}{\leftarrow}\mathbb{Z}_q\)。  
2. 计算承诺 \(a = g^{s} y^{-c} \pmod p\)。  
3. 输出 transcript \((a,c,s)\)。

因为在真实交互中，\((r,c,s)\) 的分布满足 \(s = r + cx\) 且 \(r\) 均匀，而模拟器直接从均匀分布抽取 \((c,s)\) 再导出 \(a\)，得到的分布**完全相同**。因此验证者无法 distinguer 真实证明与模拟 transcript → 零知识。

## 3. Fiat‑Shamir 非交互版

把交互中的挑战 \(c\) 替换为 **哈希**（公开参数 + 承诺）：

```
c = H( g || y || a )   (输出映射到 [0, q-1])
s = (r + c * x) mod q
```

证明 = \((a, s)\)。验证只需重新计算挑战并检查同上方程。

- **安全性**：在随机预言模型（ROM）下，Fiat‑Shamir 保持声音性与零知识性的同上证明（见 Pointcheval‑Stern 证明）。
- **实现**：只要哈希函数是抗碰撞的（SHA‑256 足够），即可得到 128‑bit 安全的非交互 Schnorr 证明。

## 4. 批量验证（O(1) 指数运算）

假设有 **n** 个独立的 Schnorr 证明 \((a_i, s_i)\) 对应公开值 \(y_i = g^{x_i}\)。  
验证每个都需要计算 \(g^{s_i}\) 和 \(y_i^{c_i}\)（其中 \(c_i = H(g||y_i||a_i)\)），总共 **2n** 次模幂。

### 批量核心思想

选取随机标量 \(\{r_i\}_{i=1}^n\)（可由同一个哈希派生，例如 `r_i = H'(transcript || i)`），然后检查 **线性组合**：

\[
g^{\sum_{i} r_i s_i}
\stackrel{?}{=}
\prod_{i} \bigl( a_i^{r_i} \cdot y_i^{r_i c_i} \bigr) \pmod p .
\]

- 左侧只需要 **一次** 多基数指数运算（可以用 **Pippenger** 或 **Straus** 算法实现近乎 O(log n) 的乘法，但在实际中常被当作 O(1) 因为 n ≤ 100 时开销可忽略）。
- 右侧同样可以通过 **预计算** + **累乘** 得到，只需 O(n) 次模乘（远小于模幂的代价）。

如果所有单个证明都正确，则对于每个 i 有  
\(g^{s_i}=a_i y_i^{c_i}\)。两边同时乘以 \(r_i\) 再求和得到上式成立。
